# 생활 약속 기록 — Google 시트 서버 저장 설정

기록을 이 기기(브라우저)뿐 아니라 **내 Google 스프레드시트에도 저장**하도록 연결하는 방법입니다.
한 번만 설정하면 폰을 바꾸거나 다른 기기에서 열어도 "지금 동기화"로 기록을 이어서 쓸 수 있고,
시트에서 전체 기록을 바로 열람·인쇄할 수 있어 상담 자료로 쓰기 좋습니다.

전체 과정은 10분 정도 걸립니다. 비용은 들지 않습니다.

## 1단계. 스프레드시트 만들기

1. [sheets.google.com](https://sheets.google.com) 접속 → **새 스프레드시트** 만들기
2. 이름은 자유롭게 (예: `생활약속기록`)

## 2단계. Apps Script 붙여넣기

1. 방금 만든 시트에서 메뉴 **확장 프로그램 → Apps Script** 클릭
2. 편집기에 기본으로 있는 코드를 모두 지우고, 이 저장소의 **`google-apps-script.gs`** 파일 내용을 전부 복사해 붙여넣기
3. 코드 위쪽의 아래 줄을 찾아 **나만 아는 비밀 코드**로 바꾸기 (영문·숫자 조합 권장)

   ```
   var TOKEN = '여기에-비밀코드-입력';
   ```

4. 저장 (💾 아이콘 또는 Ctrl+S)

## 3단계. 웹 앱으로 배포

1. 편집기 오른쪽 위 **배포 → 새 배포** 클릭
2. 톱니바퀴(⚙) → 유형에서 **웹 앱** 선택
3. 설정:
   - 다음 사용자 인증 정보로 실행: **나**
   - 액세스 권한이 있는 사용자: **모든 사용자**
     (URL과 비밀 코드를 아는 사람만 사용할 수 있으므로, 비밀 코드를 아무에게도 알려주지 마세요)
4. **배포** 클릭 → 권한 요청이 뜨면 내 계정 선택 후 허용
   ("확인되지 않은 앱" 경고가 나오면 **고급 → 이동**을 눌러 진행 — 내가 직접 만든 스크립트라 안전합니다)
5. 발급된 **웹 앱 URL**(`https://script.google.com/macros/s/…/exec`)을 복사

## 4단계. 앱에 연결

1. 생활 약속 기록 앱을 열고 [오늘 입력] 탭 맨 아래 **설정** 펼치기
2. **서버 저장 (Google 시트 연동)** 칸에:
   - Apps Script 웹 앱 URL: 3단계에서 복사한 URL 붙여넣기
   - 비밀 코드: 2단계에서 정한 비밀 코드 입력
3. **지금 동기화** 클릭 → "동기화 완료"가 뜨면 성공.
   이 기기에 있던 기존 기록이 시트에 자동으로 올라갑니다 (시트의 `기록` 탭에서 확인).

## 다른 기기에서 이어 쓰기

새 폰이나 PC에서 앱을 열고 4단계만 반복하면 됩니다 — 같은 URL과 비밀 코드를 입력하고
"지금 동기화"를 누르면 시트에 있던 기록을 모두 내려받습니다.

## 동작 방식과 알아둘 점

- 저장/삭제할 때마다 시트에도 자동 반영되고, 앱을 열 때와 자녀를 전환할 때 자동으로 동기화합니다.
- 오프라인일 때 저장한 기록은 이 기기에는 남고, 다음에 "지금 동기화"를 누르면 시트로 올라갑니다.
- 같은 날짜를 두 기기에서 각각 수정한 경우 **나중에 수정한 쪽**이 남습니다.
- 시트에서 행을 직접 지우면 그 기록은 삭제된 것으로 처리되지만, 아직 동기화 전인 기기에 남아 있으면
  그 기기가 동기화할 때 다시 올라올 수 있습니다. 삭제는 되도록 앱에서 해주세요.
- 자녀 기록이 담기므로 시트를 다른 사람과 공유할 때는 열람 범위에 주의하세요.

## 문제 해결

- **"동기화 실패: unauthorized"** — 앱의 비밀 코드와 스크립트의 `TOKEN` 값이 다릅니다. 동일하게 맞춰주세요.
- **스크립트 코드를 수정한 경우** — 반드시 **배포 → 배포 관리 → 연필(✏) → 새 버전 → 배포**를 해야 반영됩니다.
  (저장만 하면 웹 앱에는 적용되지 않습니다)
- **URL을 잃어버린 경우** — Apps Script 편집기에서 **배포 → 배포 관리**에 웹 앱 URL이 있습니다.
