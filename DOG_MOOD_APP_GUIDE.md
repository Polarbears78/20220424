# 강아지 얼굴 사진 기분 분석 앱 만들기 가이드

이 저장소에 바로 실행 가능한 **프로토타입 웹 앱**(`dog_mood_app.html`)을 추가했습니다.

## 1) 빠르게 실행하기

브라우저에서 `dog_mood_app.html` 파일을 열면 됩니다.

## 2) 현재 프로토타입 동작

- 사진 업로드
- 이미지의 밝기/채도/디테일 값을 계산
- 룰 기반으로 기분 레이블(신남/기분 좋음/차분/졸림) 출력

> 정확한 감정 추정 모델이 아니라 아이디어 검증(PoC)용입니다.

## 3) 실제 서비스로 고도화하려면

1. **강아지 얼굴 감정 데이터셋 확보**
   - 예: 행복/스트레스/불안/졸림 등 라벨 설계
2. **얼굴 검출 모델 도입**
   - YOLOv8, MediaPipe, OpenCV DNN 등으로 얼굴 영역 추출
3. **감정 분류 모델 학습**
   - TensorFlow/PyTorch로 분류기 학습
4. **앱 연동**
   - 프론트: 업로드 UI
   - 백엔드: 모델 추론 API(FastAPI 등)
5. **검증/모니터링**
   - 품종/각도/조명별 성능 평가

## 4) 다음 추천 작업

- 업로드한 사진에서 강아지 얼굴만 자동으로 크롭
- "활발함 점수"와 함께 신뢰도(confidence) 표시
- 기록 기능(날짜별 기분 추이)
